Learn to Code HTML and CSS

HTML, Hypertext Markup Languange, gives content stucture and meaning by defining that content as for example headings, paragraphs and images.
CSS or Cascading Style Sheets is a presentation Languange created to style the appreance of content---using, for examples, fonts or color.

The two Languange---HTML and CSS are independent of one another and should be remain that way.CSS should not be written inside of an HTML document and vice versa. As a rule, HTML will always represent content and CSS will always represent the appreance of the content.
