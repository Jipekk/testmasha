background-repeat: repeat (repeat X)

Here, the background image is repeated horizontally. The last image will be clipped if it does not fit. This is default:

background-repeat: repeat (repeat Y)

Here, the background image is repeated vertically. The last image will be clipped if it does not fit. This is default

background-repeat: space

Here, the background-image is repeated as much as possible without clipping. The first and last images are pinned to either side of the element, and whitespace is ditributed evenly between the images.

Hello World

background-repeat: round

Here, the background-image is repeated and squished or stretched , to fill the space (no gaps):
